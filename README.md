# xkbmapd
Based on [original](http://www.tataranovich.com/public/scripts/xkbmapd.py) script by Andrey Tataranovich and [modifified](https://bugs.launchpad.net/ubuntu/+source/xorg/+bug/1243642/comments/6) by HÃ©lio Nunes

```
usage: xkbmapd [-h] [-d] [-f file] [-p pause]

Would set your xkb keyboard layout permanent in Gnome Shell and Unity.

optional arguments:
  -h, --help  show this help message and exit
  -d          daemonize
  -f file     file to compile
  -p pause    pause before running a command
```
